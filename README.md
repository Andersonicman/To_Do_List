# To-Do List (JavaScript)

Bu proje, **saf JavaScript** kullanılarak geliştirilmiş basit bir **yapılacaklar listesi** uygulamasıdır.

## Özellikler

* Yeni görev ekleme
* Görev silme
* Görevlerin otomatik olarak numaralandırılması
* Boş görev eklenmesini engelleyen kontrol

## Kullanılan Teknolojiler

* HTML
* JavaScript

## Projenin Amacı

Bu proje, temel JavaScript konularını pekiştirmek amacıyla geliştirilmiştir:

* Diziler (Arrays)
* Fonksiyonlar
* Döngüler
* DOM manipülasyonu
* Temel kullanıcı etkileşimi

Proje, ilerleyen aşamalarda yeni özellikler eklenerek geliştirilecektir.
